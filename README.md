# Housing Market Analysiss Dashboard
## Project Overview
This project analyzes housing market data using:
- Python (Pandas)
- SQL (MYSQL)
- Power BI

## Objectives
- Data Cleaning
- Exploratory Data Analysis
- Buisness Insights
- Interactive Dashboard Creation

## Key Insights
- House area and price show a positive correlation.
- Furnished houses have higher average prices.
- More parking spaces generallly increase house prices.
- air-conditioned houses tend to be more expensive.

## Tools Used
- Python
- Pandas
- MYSQL
- POWER BI
